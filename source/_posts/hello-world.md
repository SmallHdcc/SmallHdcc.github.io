欢迎来到 [Hexo](https://hexo.io/) 这是你的第一篇推文，阅读 [文档](https://hexo.io/docs/) 来获得更多细节。如果你使用Hexo时遇到了任何问题，你可以在 [troubleshooting](https://hexo.io/docs/troubleshooting.html) 找到回答 或者在 [GitHub](https://github.com/hexojs/hexo/issues) 提问
## 快速开始

### 创建一篇新的博客

``` bash
$ hexo new "My New Post"
```

更多信息: [Writing](https://hexo.io/docs/writing.html)

### 启动服务

``` bash
$ hexo server
```

更多信息: [Server](https://hexo.io/docs/server.html)

### 生成静态文件

``` bash
$ hexo generate
```

更多信息: [Generating](https://hexo.io/docs/generating.html)

### 部署远程站点

``` bash
$ hexo deploy
```

更多信息: [Deployment](https://hexo.io/docs/one-command-deployment.html)
